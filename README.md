# docker-webex
Docker Webex - Run webex enable browser with pulseaudio enable capabilities.

# docker build command
## Inside the repository execute the command on shell

```bash
docker build --tag=webex .
```

# run browser
## Execute the command on shell

```bash
./webex.sh
```
